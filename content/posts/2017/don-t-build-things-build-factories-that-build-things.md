title: Don't build things; build FACTORIES that build things
---

There are examples of this in real world: Elon Musk ventures jump to mind.  

This idea transfers to self-dev: don't build/chase "goals"/targets; instead build the things which themselves "build" those things. I.e., systems / habits.  
  
Furthermore, there is the idea of "HabitForge"... go meta, develop a process for building habits. I.e., in essence a factory of factories.
